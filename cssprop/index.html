CSS Properties

This webpage displays al the CSS properties used till date

Category CSS Prop What it does
Color Property color Sets the color of the text
Background and Border Properties background Sets the color of all the elements
background-color Sets the background color of a single element
border Sets all the border properties for all the elements
border-radius Sets the all 4 border radius of the element
border-width Sets the width of all the borders of the element
box-shadow Gives a 3D effect to the element
Box Properties clear Restrict sides where floating elements are not allowed
display Tells how a certain HTML element would be displayed
float Specifies the position where an element should be positioned
height Sets the height of an element
width Sets the width of an element
margin Sets all the margin properties
margin-bottom Sets the bottom margin properties
margin-left Sets the left margin properties
margin-right Sets the right margin properties
margin-top Sets the top margin properties
max-height Sets the maximum height of an element
max-width Sets the maximum width of an element
min-height Sets the minimum height of an element
min-width Sets the minimum width of an element
padding Sets all the padding properties
padding-left Sets the left padding properties of element
padding-right Sets the right padding properties of element
padding-top Sets the top padding properties of element
padding-bottom Sets the bottom padding properties of element
z-index Sets the stack order of a positioned element
Text Properties text-align Specifies the horizontal alignment of text
line height Sets the line height
Text Decoration Prperties text-decoration Specifies the decoration added to text
text-decoration-color Specifies the color of the text-decoration
Font Properties text-decoration Sets all the font properties in one declaration
font-family Specifies the font family for text
font-size Specifies the font size of text
font-style Specifies the font style for text
font-weight Specifies the weight of a font
Transition Properties transition-timing-function Specifies the speed curve of the transition effect
transition-delay Specifies when the transition effect will start
List Properties list-style Sets all the properties for a list
Marquee Properties marquee-direction Sets the direction of the moving content
marquee-play-count Sets how many times the content move
marquee-speed Sets how fast the content scrolls
marquee-style Sets the style of the moving content
